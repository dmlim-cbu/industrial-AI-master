- [02] ROS node & topic (하나의 node에서 2개이상의 토픽 subscribe하기)
  - 과제 내용 : 
    - 노드는 다음과 같이 구성한다.
    - 1번 node : 1과 10 사이의 랜덤한 값을 A로 publish
    - 2번 node : 11과 20 사이의 랜덤한 값을 B로 publish
    - 3번 node : A와 B를 subscribe한 후, A+B를 연산하여 C로 publish
    - 4번 node : C subscribe 누적 평균 출력
